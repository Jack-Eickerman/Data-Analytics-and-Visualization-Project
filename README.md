# Supply Chain Data Cleaning Pipeline (Python)

## Overview
This project implements a **data cleaning and preprocessing pipeline** for a simulated supply chain dataset using Python and the Pandas library.  

The objective is to:
- Identify and handle missing values
- Detect and remove outliers
- Standardize and transform data
- Produce a clean dataset suitable for analysis and modelling

---

## Technologies Used

- **Python 3.x**
- **Pandas** – data manipulation and analysis
